SkyClient installs things like mods and texture packs for Hypixel into a Minecraft Forge profile (those things in the Minecraft launcher).

Cloudclient is a version of SkyClient, accessible online.

Your browser doesn't have any way to access files on your computer. This means that Cloudclient can't create your profile. Try again with Google Chrome, Microsoft Edge, or any other Chromium-based browser.

      Installing

      What Minecraft launcher would you like to install for?

      MultiMC Prep

      If you use MultiMC, first you need to create an instance for SkyClient.

      1. Click the "Add Instance" button
      2. Name the instance SkyClient and add it to a group if you want
      3. Set the Minecraft version to 1.8.9
      4. Set the icon to the SkyClient logo (click to download)
      5. Select the instance and click "Edit Instance"
      6. Click "Add Forge"

      Then, when you upload your .minecraft folder, choose the one you find from the "Open .minecraft" button.


      Now you need to let Cloudclient download and add mods, texture packs, and config files to your .minecraft folder. You'll need to find it and put it here.

      Drop here
      Or click to browse

      By continuing, you agree to let Cloudclient modify your .minecraft folder, and acknowledge that it may overwrite existing mods, texture packs, or configuration. Cloudclient, SkyClient, and our affiliates are not liable if something goes wrong.

      Taking a backup is highly recommended. You can always close this tab or reload the page.

      Consider removing existing mods/texture packs/config in the directory you chose.
